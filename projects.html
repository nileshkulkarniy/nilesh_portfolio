Attendance Management System

Java Web Application

Stack: Java, JSP, Servlets, MySQL, JDBC, HTML, CSS, JavaScript, Maven, Jetty

Developed a comprehensive Attendance Management System for educational institutions using Java Servlets, JSP, and MySQL. Implemented role-based access with separate Admin and Teacher portals. Admin Module manages courses, subjects, students, teachers, schedules, and generates attendance reports. Teacher Module allows marking daily attendance and viewing class-wise, subject-wise, and monthly attendance reports.

GitHub Repo

Key features

  • Role-based access with separate Admin and Teacher portals
  • Admin Module: Manage courses, subjects, students, teachers, schedules
  • Teacher Module: Mark daily attendance, view reports
  • 3-tier architecture (Presentation, Business Logic, Data Layer)
  • Session-based authentication and secure password handling
  • Responsive UI with HTML, CSS, JavaScript, and AJAX
  • Built and deployed using Maven with embedded Jetty server
